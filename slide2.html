Princípio Central

Dependências sempre apontam para dentro. Camadas externas conhecem as internas, mas não o contrário.

Responsabilidades das Camadas

Controllers

  • • Gerenciam requisições HTTP
  • • Validam dados de entrada
  • • Formatam respostas

Services

  • • Implementam regras de negócio
  • • Coordenam operações
  • • Processam dados

Models

  • • Definem estruturas de dados
  • • Representam entidades
  • • Interagem com banco de dados
