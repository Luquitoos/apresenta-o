Integração com Backend

Services Integrados

authService.js
• login() → POST /auth/login
• register() → POST /auth/register
• logout() → POST /auth/logout
todoService.js
• getTodos() → GET /todos
• createTodo() → POST /todos
• updateTodo() → PUT /todos/:id
• deleteTodo() → DELETE /todos/:id
• searchTodos() → GET /todos/search
